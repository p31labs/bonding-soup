# Security Policy

## Supported Versions
| Version | Supported |
| ------- | --------- |
| main | ✅ Active development |

## Reporting a Vulnerability
Report via GitHub Security Advisories: https://github.com/p31labs/bonding-soup/security/advisories/new

## Security Practices
- CI workflows use pinned SHA-based action versions
- Workflow permissions set to read-only by default
- Branch protection requires PR review and passing status checks
