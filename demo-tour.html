A guided walk · about seven minutes

Come in. Let me show you around.

This is P31 Labs — a small lab building free, open-source tools for families whose brains work differently. Take your time. Nothing here is trying to capture your attention. We are just trying to be useful, and to show our work.

  1. The Room

    A shared space for your people

    C.A.R.S. is a small room where families can play together — dragging atoms, building molecules, watching the colors breathe. No strangers can wander in. There is no leaderboard.

    Most “together” software is actually a stadium. This one is a living room.

    Enter the room

    When you arrive: drag an atom from the palette, build water (H + H + O), and watch the room respond.

  2. Cognitive Passport

    A one-page summary of how your brain works

    A short, plain-English document you fill out for yourself. Then you can hand the same one page to a doctor, a teacher, or a family member, instead of explaining everything from scratch.

    Saying the same things over and over is one of the most expensive things a neurodivergent person does. This makes that cheaper.

    Open the passport

    Pick an audience, fill in one section, then export it.

  3. Document library

    The receipts, all searchable

    Every plan, standard, and specification we use to build this lab lives here — in plain markdown, fully searchable in your browser. Nothing is hidden behind a portal.

    “We build in public” has to mean something. This is what it means.

    Search the library

    Type a word like legal or mesh in the search box. Watch what comes back.

  4. Fleet portal

    Every live URL on one page

    A single page that lists everything we have running, with a small dot showing whether it is up. The list is generated, not hand-typed, so it tells the truth.

    If a thing is not on this page, it is not real yet.

    Open the fleet portal

    Count the green dots. That is what is running right now.

  5. Device setup

    How to run all of this on a Chromebook and a phone

    The same instructions we use ourselves. A Chromebook on the desk, a phone in your pocket, and a small set of commands — nothing fancier than that.

    The lab should run on the hardware a real family already owns.

    Read the setup

    Scroll through. This is the operator’s field manual.

  6. Command center

    The control panel — locked by default

    A small page on this computer with a list of buttons. Each button runs one specific, pre-approved action. Nothing else can run from here.

    A control panel that lets anything run is not a control panel. This one is gated, on purpose.

    Open the control panel

    Click Doctor in Essentials. Watch it run, then come back.

  7. Operator desk

    Read-only telemetry — the calm window

    The desk does not run anything. It only shows the current state — connection counts, glass probes, what is up, what is quiet. Leave it open in a second tab.

    Two windows: one for action, one for awareness. They should never be the same window.

    Open the desk

    Keep this open while you finish the tour. Glance over once or twice. That is the whole job.

The library search is the fastest way to prove this is real without explaining anything. Type a word and read what comes back.
