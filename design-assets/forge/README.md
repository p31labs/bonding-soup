# Apex Forge (Sovereign Asset Forge v2)

Open **`apex-forge.html`** locally in a browser (e.g. from repo root: `python3 -m http.server 8080` and open the path, or open the file directly). Click panels to copy raw SVG payloads to the clipboard.

**Includes:** Larmor kinetic logo, biometric-style PFP block, tomography/OpenGraph-style banner.

**Dependencies:** Tailwind CDN, Google Fonts (Atkinson / JetBrains / Playfair). No build step.

**Next integration:** Optional publish to `p31ca.org` as static `asset-forge.html` via existing passport-style deploy path; wire spinners from `../spinners/` for shared loading chrome.
