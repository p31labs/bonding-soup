# Ko-fi Asset Engine

Open **`kofi-engine.html`** locally. Click regions to rasterize SVG to PNG via canvas (download). Dimensions: header 1200×600, product tiles 600×600.

**Ethical alignment:** Treat product titles (“Super-Centaur Pack”, “Node Zero Firmware”, “Operator Grant”) as **draft labels** — confirm against `docs/ETHICAL-STYLE-MAP.md`, `docs/P31-CREATE-CONNECT-ETHICAL-MONETIZATION.md`, and published shop data before uploading to Ko-fi.

**Note:** Canvas text rendering depends on loaded web fonts; if PNG text looks wrong, use OS screenshot on the SVG preview or bake outlines in a design tool.
