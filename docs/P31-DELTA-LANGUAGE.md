## 1. Purpose

DELTA is a **small, stable vocabulary** for describing how P31 builds feel — one institution, one sky, many altitudes. It is **easy to learn** (thirty-ish words, four anchors) and **deliberately unfinishable** in the sense that music or poetry is: you never “graduate” from caring how transitions feel.

**DELTA** doubles as a mnemonic: **Differential Embodied Language of Threshold and Affect**. The Greek letter Δ still means *change* in math; here it also names **closed loop** and **return** — the resilient path from Wye stress to Delta closure on [delta.html](https://p31ca.org/delta) and in real wiring.

This document is **normative prose**. The JSON registry is the **machine anchor** for verifiers and tools; if prose and JSON disagree on a term ID, **fix prose or JSON in the same commit**.

## 2. Four anchors (non-negotiable)

These three phrases plus one rhythm define whether copy is “in family”:

| Anchor | Meaning |
|--------|--------|
| **Same void** | Layer 1 is one institution’s quiet floor — Gray Rock, no performative brand on first paint. |
| **Same sky** | One night field; teal / coral / hearth are one family; moving between seeds is **altitude**, not product-switching. |
| **No jarring cut** | Transitions feel like walking rooms in one building — ribbon, starfield stratum, typography stay kin. |

The fourth anchor is **rhythmic**: **verify before velocity** where facts are claimable — not vibe-substitute for the ship bar.

## 3. Layers and rooms

| Term | Definition |
|------|------------|
| **void** | Undifferentiated ground — background, not noise. |
| **sky** | Shared visual field across surfaces; not a component, the hemisphere you navigate. |
| **Layer 1** | Void / Gray Rock — default calm. |
| **Layer 1.5** | Starfield stratum — luminance atmosphere between void and Alive; weather, not logo. |
| **Layer 2** | Alive — interaction, focus, handoff; still one building. |
| **altitude** | How much of the system you see — hub wide, bond tight; not a different product. |
| **window** | Any seed or page — viewport onto the same sky. |
| **room** | Coherent affordance space with a **host mind**; you walk, you don’t teleport. |
| **walk** | Transition that preserves sky/void continuity. |
| **jarring cut** | When the user feels they changed apps — fix with shared sky, ribbon, motion discipline. |

## 5. Topology (shared with electrical and mesh canon)

| Term | Definition |
|------|------------|
| **Wye (Y)** | Branchy stress — one center, three arms; open, divergent. |
| **Delta (Δ)** | Closed three-phase loop — resilient run; operator return path. |
| **K₄** | Complete graph on four vertices — the cage; family mesh. |
| **edge** | Relationship segment — load path, not decoration. |
| **vertex** | Person, pillar, or dock — not “user node” in growth-hack copy. |
| **mesh** | Family/community substrate — honest latency, not omniscience. |
| **cage** | K₄ family container — explicit boundaries; never panopticon metaphors. |
