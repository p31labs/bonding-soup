## Project Description

### Node One (Totem)

Node One is a handheld assistive device designed for neurodivergent operators who need:
- Somatic regulation through precise haptic feedback (DRV2605L LRA motor driver)
- Private communication without surveillance-capitalism data models (LoRa SX1262 mesh)
- Secure device identity (NXP SE050 secure element)
- Long battery life for all-day carry (ESP32-S3 ULP coprocessor)
