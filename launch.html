\u00a702 \u00b7 the launch verbs

Three commands. The whole launch is in npm run launch. Add --dry-run to see what it would do; add --status to read the last result without re-running.

npm run launch

Canonical assembly: verify (83 gates) \u2192 build PWA + demos + fleet portal \u2192 apply constants \u2192 sync passport \u2192 strict mesh + hub:ci + security \u2192 rewrite this dashboard. Idempotent. Local only. Set P31_LAUNCH_PUBLISH=I_UNDERSTAND to also push + deploy.

npm run launch -- --dry-run

Read-only preview. Tells you what would happen, without touching anything.

npm run launch -- --status

Prints the last saved readiness JSON. No gates re-run, no writes. Safe to call from cron, command-center, or before-coffee.
