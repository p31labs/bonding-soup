§1 · If you remember nothing else — the five commands

Spoon-deficit mode: pick one, run it, that's the win. Every other section assumes you have more bandwidth. These five do not.

1. Verify the ship bar npm run verify

~84 gates. If green, the geometry holds. If red, follow the failing line — it points to the fix.

2. Full assembly + rainbows npm run launch -- --full

~85s warm. 31 steps. Every deliverable + every probe. Rainbow finale on green. Doesn't deploy.

3. Local control plane npm run command-center

Browser at :3131. Every whitelisted action one click away. Operator desk at /desk.

4. Health snapshot p31 doctor

Remotes, ports, key files. Add --verify for the gate. Add --mesh for live edge.

5. Make a PR npm run pr

Auto-branch, commit, push, gh pr with auto-merge if on main. Stops if your local is dirty in the wrong place.

The rainbow fires on assembly completeness, not probe completeness. Command-center down on this machine ≠ ship not ready. Probes are informational.
