Full setup checklist

Run in order. If npm run verify disagrees with a step, fix the environment first.

  1. Chrome OS current stable; reboot after large updates.
  2. Developers → Linux — install Crostini; resize disk for Node + node_modules (and Andromeda if present).
  3. Node 20+ in Linux — see repo .nvmrc and AGENTS.md.
  4. Clone home repo into Linux home, e.g. ~/p31. Andromeda is a second clone beside it when needed — P31-ROOT-MAP.md.
  5. One-shot — npm run setup from repo root (install, soup prep, apply constants / style, verify).
  6. Bookmarks — local demo, p31ca.org, BONDING Soup (live), this page.
npm run setup

Optional daily rhythm: npm run morning (pull, converge, command center) — when you have adopted that script.

Acer Spin 7xx (typical)

2-in-1 — use tablet mode for long reads (doc library, docs/); clamshell + power for builds.

Stylus (USI) — model-dependent; not required for development.

Thermals — expect fan noise under npm run p31:all or large pnpm install; that is normal. Prefer a cool, hard surface.

Performance truth — for BONDING Soup, measure on this hardware; see SOUP-PERF-BUDGET.md.

Linux · path · static server

Always serve from the repo root

The demo script binds the directory that contains soup.html. Wrong directory = wrong site.

cd ~/p31 && npm run demo

Adjust ~/p31 to your clone. Or: npm run demo --prefix /path/to/p31.

Port 8080 busy

P31_DEMO_PORT=8090 npm run demo

Details: SOUP-LOCAL-DEMO.md.

Command center on your phone (LAN, trusted Wi‑Fi)

Whitelisted actions from the phone against your Chromebook. Same network only — not for coffee-shop Wi‑Fi.

P31_CMD_CENTER_LAN=1 npm run command-center

Then open http://<CHROMEBOOK_LAN_IP>:3131 on the phone. The UI shows a phone-friendly URL. iOS: Safari → Add to Home Screen — MOBILE-OPS-PHASE2.

hostname -I | awk '{print $1}'

Troubleshooting

SymptomWhat to do
No such file or directory after cd Use your real path — not a placeholder. ls soup.html in the folder first.
Address already in use P31_DEMO_PORT=8090 npm run demo or stop the other process on 8080.
verify OOM / killed Enlarge Crostini disk; more RAM free; MESH_LIVE_STRICT=0 for offline pass per AGENTS.md.
Phone cannot load :3131 Same Wi‑Fi, correct LAN IP, firewall; command center started with P31_CMD_CENTER_LAN=1.
