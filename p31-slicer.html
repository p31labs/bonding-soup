What to expect

Embedded UI below loads https://grid.space/kiri/. You need a network path to GridSpace unless you self-host grid-apps and swap the iframe src in this file.

K₄ tetrahedron STL in this repo: design-assets/stl/P31_K4_Topology.stl. Because the iframe is HTTPS and local demo is usually HTTP, do not expect “paste localhost URL” to import cleanly — use drag-and-drop or Kiri’s file picker on the downloaded file.

Heavy work: slicing is CPU/RAM intensive (GridSpace recommends planning for enough RAM). Respect prefers-reduced-motion in your own overlays; Kiri has its own UI settings.
