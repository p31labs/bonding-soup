How to share these

Native screenshot (cleanest) · Open this page locally with npm run demo, then load http://127.0.0.1:8080/social-cards/ on a desktop browser. Each card renders at the responsive size your screen allows; cmd/ctrl + + to zoom in for sharper screenshots, then crop to the square.

PDF export · Click ⎙ print all (PDF) in the toolbar. Each card prints on its own page, suitable for grant-application appendices or attaching to a PR.

Direct edit · Open social-cards/index.html in your editor. Numbers, copy, and palette are all inline — no build step. Update once, regen all share assets.
